PC 환경에서만 실행 가능합니다.  
https://github.com/kminito/srt_reservation 를 참고했습니다.

  
## 필요

- selenium
- webdriver_manager
- fastapi
- uvicorn[standard]
- jinja2
- python-multipart


## 사용법
> 가상 환경을 만드는 것을 추천합니다.

```py
1. python -m venv .venv
2. source .venv/bin/activate or ./.venv/scripts/activate.ps1
3. pip install -r requirements.txt
4. python app.py
5. localhost:8000 접속
```



## 기타  
명절 승차권 예약에는 사용이 불가합니다.
